ControlWastedBustedScreen

ASI plugin for GTA San Andreas classic PC.

Installation:
1. Copy ControlWastedBustedScreen.asi and ControlWastedBustedScreen.ini into your GTA San Andreas folder.
2. Make sure you have an ASI loader installed.
3. Edit ControlWastedBustedScreen.ini to customize timings.

All values are in milliseconds.

Example:
1000 = 1 second
500 = half a second

This plugin was designed for GTA SA classic PC 1.0 US.
